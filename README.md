# Image-to-String-tool

同济大学大一高程大作业——将图像转化字符串组成的画面，支持图片和视频转化。

主要原理是不同的字符串对应不同深浅的颜色，最后平均每张图片转化需要500~600ms，个人感觉是一个不错的图像处理的入门项目。
